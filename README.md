# 🍰 AuroBakery · Artisanal Bakes

A premium, modern **static website** for AuroBakery—a contemporary Indian bakery café showcasing handcrafted breads, artisanal cakes, and fine pastries. Built with clean HTML and CSS, optimized for accessibility and visual excellence.

---

## 📖 Overview

**AuroBakery** is a multi-page website portfolio for a specialty bakery and café located in Bengaluru, India. The site presents the brand's premium positioning through:

- 🎯 Elegant hero section with featured offerings
- 📋 Comprehensive menu with Indian pricing (₹)
- 🏪 Store information and contact details
- 📱 Fully responsive design
- ♿ Semantic HTML structure

**Tech Stack**: HTML (77.2%) + CSS (22.8%)  
**No Build Tools Required** – Open directly in any modern browser

---

## 🚀 Quick Start

### Option 1: Direct Browser (Easiest)
Simply double-click `index.html` to open the site locally. No server needed.

### Option 2: Local Development Server

**Using Python 3:**
```bash
python -m http.server 8000
```
Then visit: `http://localhost:8000`

**Using Node.js (http-server):**
```bash
npx http-server
```

**Using VS Code Live Server Extension:**
Install the Live Server extension and right-click `index.html` → "Open with Live Server"

## 🔧 Customization Guide

### Update Business Info
Edit these files to add your real details:

**In every `.html` file:**
- Phone: `+91-98765-43210`
- Email: `hello@aurobakery.in`
- Address: `Indiranagar, Bengaluru`
- Hours: `8:00 AM – 10:30 PM`

### Modify Menu & Pricing
Open `menu.html` and update:
- Product names, descriptions, and prices
- Dietary tags (Vegan, Eggless, etc.)
- Category sections

### Customize Styling
Edit `assets/css/style.css`:
- **Colors**: Update CSS variables for brand palette
- **Fonts**: Modify Google Fonts imports in HTML `<head>`
- **Spacing**: Adjust padding/margin values
- **Breakpoints**: Responsive media queries

### Replace Images
1. Find Unsplash image URLs in HTML files
2. Replace with your own product photography
3. Update `alt` text for accessibility

Drag & drop the project folder to Netlify for instant hosting.

Upload all files via FTP to your web host—no build step needed!
