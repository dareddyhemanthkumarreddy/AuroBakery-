# AuroBakery · Static Multi‑Page Website

This is a fully static, multi‑page bakery website intended to look modern and premium. It can be opened directly in a browser (no build tools required).

## Pages

- **`index.html`**: Landing page with hero section and featured bakery items.
- **`menu.html`**: Full bakery menu with many cakes, pastries, breads, cookies and add‑ons listed in Indian Rupees (₹).
- **`about.html`**: Brand story, philosophy, and team highlights.
- **`contact.html`**: Contact details, visit information, and a demo enquiry form.

All pages share a common navigation bar and footer, styled via `assets/css/style.css`.

## Running the site

- **Option 1 (simplest)**: Double‑click any of the `.html` files (for example `index.html`) to open it in your browser.
- **Option 2 (local server)**:
  - Using Python:
    - Python 3: `python -m http.server 8000`
  - Then visit `http://localhost:8000` in your browser.

## Customisation

- Update text and prices directly in the relevant `.html` files.
- Adjust colours, spacing, and layout in `assets/css/style.css`.
- Replace any placeholder copy (e.g. address, phone, email) with your real details.
